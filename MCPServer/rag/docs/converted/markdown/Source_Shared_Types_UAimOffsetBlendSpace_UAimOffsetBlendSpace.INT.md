## Content

### UAimOffsetBlendSpace

An Aim Offset is an asset that stores a blendable series of poses to help a character aim a weapon. During animation, the result of the Aim Offset is blended with other motions such as running, walking, jumping, etc. to cause the character to appear to smoothly look in all directions.
