## Content

### ADefaultPawn

The DefaultPawn class contains a native DefaultPawnMovementComponent, a spherical CollisionComponent, and a StaticMeshComponent. 
The movement style set for the DefaultPawnMovementComponent is a no-gravity, flying style. In addition to the usual MovementComponent variables, it includes MaxSpeed, Acceleration, and Deceleration float values.
