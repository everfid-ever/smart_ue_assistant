## Content

### ActionMappings

Allows you to map discrete button or key presses to a "friendly name" that will later be used to trigger events.

### AxisMappings

Allows you to map keyboard, controller, or mouse inputs to a "friendly name" that will later be used for continuous game behavior (eg. movement).

### AxisConfig

Allows you to set up the properties of (hardware) input axis.
